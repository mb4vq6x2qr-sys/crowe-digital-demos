_snippets/ — Wiederverwendbare Code-Blöcke
==========================================

Jeder Snippet hat einen eigenen Ordner mit:
  snippet.html   → fertiger Vanilla HTML/CSS/JS Code
  info.md        → Beschreibung, Verwendungszweck, Branchen, Anpassungen

Keine Branchen-Unterordner mehr — Claude liest die info.md
und wählt / empfiehlt passende Snippets für jedes Projekt.

Snippet hinzufügen
------------------
1. TSX/JSX Code hier in den Chat kopieren
2. Claude konvertiert zu snippet.html
3. Claude erstellt info.md mit Beschreibung
