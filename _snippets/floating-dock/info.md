# Floating Dock

**Was:** macOS-Dock-Effekt: Icon-Leiste die fixiert am unteren Bildschirmrand schwebt. Icons vergrössern sich wenn der Cursor nähert (Federsimulation). Mobile: Aufklapp-Button unten rechts.
**Typ:** Navigation
**Passt für:** alle Branchen
**Ideal für:** moderne One-Page-Sites, alternative oder ergänzende Navigation, Seiten mit flottem UI-Stil
**Stimmung:** modern, verspielt, interaktiv, premium
**Nicht ideal für:** klassische/konservative Seiten, Seiten mit vielen Nav-Punkten (>6), Mobile-heavy ohne gute Desktop-Version
**Anpassen:** `ITEMS` Array im `<script>` (title, href, icon als SVG-String), `MIN_SIZE`/`MAX_SIZE` für Grössenverhältnis, `--fd-bg`/`--fd-icon-bg` CSS-Variablen für Farben
**Hinweis:** Wird via `position: fixed` positioniert — funktioniert unabhängig vom umgebenden HTML
