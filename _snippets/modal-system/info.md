# Modal System

**Was:** CSS + JS Basis für alle Modals auf einer Seite. Einmal einbinden, beliebig viele Modals verwenden. Schliesst per X-Button, Klick ausserhalb, oder Escape.
**Typ:** Infrastruktur
**Passt für:** alle Branchen
**Ideal für:** Impressum, Datenschutz, Stornierung, Bildergalerie, Produktdetails
**Immer einbinden wenn** `impressum-ch`, `datenschutz-ch` oder `stornierung` verwendet werden
**Abhängigkeit:** keine
