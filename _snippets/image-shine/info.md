# Image Shine Hover

**Was:** Diagonaler weisser Schimmer gleitet beim Hover über ein Bild (wie Lichtreflex auf Glas). Optional leichter Zoom-Effekt.
**Typ:** UI-Effekt
**Passt für:** alle Branchen
**Ideal für:** Service-Karten mit Bild, Team-Fotos, Galerie-Bilder, Hero-Bilder
**Stimmung:** frisch, lebendig, professionell
**Anpassen:** Opacity des Schimmers (`rgba(255,255,255,0.25)`), Übergangs-Geschwindigkeit (600ms), Zoom-Stärke (scale 1.03)
**Verwendung:** Klasse `img-shine` auf jeden Container setzen der ein `<img>` enthält
