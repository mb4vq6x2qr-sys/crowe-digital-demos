# Container Text Flip

**Was:** Wechselt Wörter mit Buchstaben-für-Buchstaben Blur-Einblend-Animation. Container-Breite passt sich automatisch an die Wortlänge an.
**Typ:** Animierter Inline-Text
**Passt für:** alle Branchen
**Ideal für:** Hero-Slogan («Fühlen Sie sich [entspannt/erholt/wohlig]»), Headline mit wechselndem Adjektiv, Angebots-Highlights
**Stimmung:** modern, dynamisch, frisch
**Nicht ideal für:** lange Wörter auf kleinen Screens, Texte die SEO-relevant sein müssen
**Anpassen:** `data-words` (JSON Array), `data-interval` (ms zwischen Wechseln), `data-duration` (Animationsdauer ms), Schriftgrösse via CSS `.ctf-container`
