# Text Hover Effect

**Was:** SVG-Text mit drei Ebenen: zeichnet sich beim Laden ein (Stroke-Animation), zeigt auf Hover einen Regenbogen-Farbverlauf der dem Cursor folgt.
**Typ:** Hero-Element / Überschrift
**Passt für:** alle Branchen
**Ideal für:** Hero-Hauptüberschrift, grosse Sektion-Titel, auffälliger Markenname
**Stimmung:** premium, auffällig, interaktiv
**Nicht ideal für:** kleine Texte, Fliesstext, Mobile-only Sites
**Anpassen:** Text 3× im HTML (gleicher Wert), `font-size` (SVG-Einheiten), Farben im `linearGradient`
**Hinweis:** Bei mehrfacher Verwendung auf einer Seite IDs (`th-*`) eindeutig machen
