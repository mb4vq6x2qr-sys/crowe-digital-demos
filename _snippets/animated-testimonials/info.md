# Animated Testimonials

**Was:** Bewertungen mit Bild-Stack links (gestapelte Fotos, aktives springt hoch) und Text rechts (Wörter blenden per Blur ein). Navigation per Pfeiltasten, optional Autoplay.
**Typ:** Bewertungen / Social Proof
**Passt für:** alle Branchen — besonders Massage, Coiffeur, Restaurant, Dienstleister
**Ideal für:** Bewertungs-Sektion, Über-uns-Ergänzung, Vertrauensaufbau
**Stimmung:** persönlich, vertrauenswürdig, lebendig
**Nicht ideal für:** Seiten ohne Kundenbewertungen oder Fotos
**Anpassen:** `testimonials` Array im `<script>` (name, designation, quote, src), `data-autoplay="true"` für automatischen Wechsel
