# Animated Modal

**Was:** Modal mit 3D-Einblend-Animation (scale + rotateX + spring). Overlay mit Blur-Hintergrund. Schliessen per X-Button, Klick ausserhalb oder Escape. Mehrere Modals pro Seite via data-modal-trigger="[id]".
**Typ:** UI-Komponente / Overlay
**Passt für:** alle Branchen
**Ideal für:** Bildergalerie-Lightbox, Buchungs-Details, Produkt-Info, Angebots-Details, Kontaktformular im Popup
**Stimmung:** modern, elegant, professionell
**Nicht ideal für:** wenn bereits modal-system verwendet wird (dann stattdessen jenes verwenden — gleiche Funktion, anderer Stil)
**Anpassen:** `data-modal-trigger="[id]"` auf Trigger-Button, `id="[id]"` auf `.am-overlay`, Inhalt in `.am-content`, `.am-footer` für Aktions-Buttons, `max-width` in `.am-box`
**Hinweis:** Ersatz oder Ergänzung zu `modal-system`. Dieser hat 3D-Animation, modal-system hat den dunkleren Spa-Stil.
