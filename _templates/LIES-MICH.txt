Was gehört hier rein?
====================

Fertige HTML-Snippets – ein File pro Seiten-Abschnitt (Section).

Jedes Template ist ein eigenständiger HTML-Block den Claude
in eine neue Website einfügen und anpassen kann.

Beispiele für Dateinamen:
  hero-gradient.html       → grosser Einstiegsbereich mit Hintergrundbild
  hero-video.html          → Hero mit Hintergrundvideo
  services-grid.html       → Leistungen als Karten-Raster (3er Grid)
  services-list.html       → Leistungen als vertikale Liste mit Icon
  team-cards.html          → Team-Sektion mit Foto, Name, Rolle
  testimonials.html        → Kundenstimmen / Bewertungen
  pricing-table.html       → Preistabelle (1–3 Pakete)
  faq-accordion.html       → Häufige Fragen mit Auf/Zu-Effekt
  contact-form.html        → Kontaktformular via Formspree
  magic-nav.html           → Mobile Bottom-Navigation mit Pop-Effekt
  footer-simple.html       → Footer mit Links und Copyright
  cookie-banner.html       → Cookie-Hinweis (DSG-konform)

Woher bekommst du Templates?
→ Selbst gebaut (aus bisherigen Demos extrahieren)
→ GitHub: "html css components vanilla" suchen
→ Aceternity UI, Uiverse.io – kopieren und anpassen

Wie benutzt du Templates?
"Verwende _templates/hero-gradient.html und services-grid.html"
Claude nimmt den Code, passt Farben/Inhalte an den Kunden an.
