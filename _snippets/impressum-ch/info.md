# Impressum (CH)

**Was:** Impressum als Modal für Schweizer Einzelfirmen.
**Typ:** Rechtliches
**Passt für:** alle Schweizer Websites
**Ideal für:** Footer-Link, auf jeder echten Website Pflicht
**Anpassen:** [FIRMENNAME], [INHABER], [STRASSE NR], [PLZ], [ORT], [TELEFON], [EMAIL], [WEBSITE]
**Abhängigkeit:** Braucht `modal-system` (CSS + JS)
